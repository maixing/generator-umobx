### 开始安装
> 首先确保自己已经安装了nodejs
### 安装yeoman
> npm install -g yo
### 然后安装脚手架
> npm install -g generator-webpack3
### 创建项目
> 新建项目目录
> 进入新建项目目录后执行yo webpack3
### 项目命令
> 先执行：`npm run dll`
> 开发模式:`npm start`
> 生产模式:`npm run build`
### mac上项目命令
> 先执行：`npm run dll`
> 开发模式:`npm run start-mac`
> 生产模式:`npm run build-mac`
